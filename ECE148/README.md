This repository contains assignments from a course on signal analysis and processing applications (ECE 148), focusing on a wide range of engineering applications. 

Topics include:
- communications systems
- image processing
- analog and digital filter design
- signal detection
- parameter estimatio
- holography, tomography, Fourier optics, and microwave and acoustic sensing

The course emphasizes applying theoretical concepts to practical problems, developing the skills needed to analyze, design, and implement signal processing techniques across diverse real-world domains.
