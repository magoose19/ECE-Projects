This repository contains assignments from a course on neural networks and deep learning (ECE 180), covering the evolution and application of modern machine learning models. Topics include foundational concepts such as perceptrons and associative memory, as well as core architectures like multilayer neural networks trained with backpropagation. The course also explores advanced models including convolutional neural networks (CNNs), recurrent neural networks (RNNs), and attention-based architectures. Through hands-on projects, students apply these techniques to real-world problems in areas such as natural language processing and computer vision, building practical skills in designing and implementing deep learning systems.
