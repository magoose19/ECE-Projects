This repository contains assignments from ECE 157C/272C (2026), which focuses on the foundations and applications of AI agents in the semiconductor industry. The course introduces core concepts of AI agents and explores their use in analyzing semiconductor test data. Building on this, assignments involve developing agents for test data analytics and extending these ideas toward hardware-oriented applications. Through a combination of theory and hands-on projects, the course emphasizes practical implementation, data-driven problem solving, and the integration of AI techniques into real-world engineering workflows.

Brief Overview:
- Introduction to AI Agent
- Introduction to Semiconductor Test Data
- Agent Building for Test Data Analytics
- Agent Building for Hardware Code
