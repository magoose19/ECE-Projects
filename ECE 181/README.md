This repository contains assignments from an introductory Computer Vision (ECE 181) course focused on enabling machines to interpret and analyze visual data. The course explores how images are formed, how sensors capture visual information, and how algorithms can extract meaningful insights from images and video. 

Topics include:
- image formation and projective geometry
- feature extraction using both traditional and modern techniques
- core methods for object recognition, detection, and segmentation

Through hands-on programming assignments, the course emphasizes building practical computer vision applications, equipping students with the skills to design, implement, and evaluate real-world vision systems.
